# atlassian-tools
Electron Toolkit for Atlassian's JIRA & Crucible

### To Run From Source:
```
npm install
npm start
```

### To Build
```
npm run build
"build": "rimraf AtlassianTools-* && electron-packager . --platform=win32,darwin,linux --arch=x64 --icon=app"
```
