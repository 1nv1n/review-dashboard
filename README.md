### To Run From Source:
```
npm install
npm start
```

### To Build
```
npm install
npm run build
```
This will create the following executable containing directories:
 - AtlassianTools-win32-x64
 - AtlassianTools-linux-x64
 - AtlassianTools-darwin-x64
